---
title: MPC Core Kit Node Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth MPC Core Kit in
your Node.js app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/mpc-core-kit/33).

:::tip building locally

- Clone the MPC Core Kit Node Node Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/tree/main/mpc-core-kit-node/mpc-core-kit-node-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm start
  ```

:::
