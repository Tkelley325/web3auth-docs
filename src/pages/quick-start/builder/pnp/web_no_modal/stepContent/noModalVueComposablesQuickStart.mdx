---
title: Web3Auth PnP No Modal Vue Composables Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth Plug and Play No
Modal in your Vue app. This app uses Vue Composables for easy configuration.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-pnp/pnp-web/7).

:::tip building locally

- Clone the PnP No Modal Vue Composables Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-pnp-examples/web-no-modal-sdk/quick-starts/vue-composables-no-modal-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm run start
  ```

:::
