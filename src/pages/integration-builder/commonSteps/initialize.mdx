---
title: Initializing
---

Finally, it is time to initialise the SDK within your application. Initialisation should happen within a constructor, this ensures there's no lag
between when the user clicks on the login button and the modal is shown. Here is where you customise your modal login buttons or can use the default
set of logins & wallets, by not passing any argument into the function.

- [Initialising Web3Auth Modal](/sdk/pnp/web/modal/initialize#initializing-modal).

:::
