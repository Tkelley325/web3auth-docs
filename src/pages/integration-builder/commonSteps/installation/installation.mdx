---
title: Install Web3Auth
---

Install `@web3auth/modal` & `@web3auth/base` using either [npm](https://www.npmjs.com/package/@web3auth/modal) or
[yarn](https://yarnpkg.com/package/@web3auth/modal):

```bash npm2yarn
npm install --save @web3auth/modal @web3auth/base
```
