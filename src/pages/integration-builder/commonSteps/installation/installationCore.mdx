---
title: Install Web3Auth
---

Install `@web3auth/no-modal` & `@web3auth/base` using either [npm](https://www.npmjs.com/package/@web3auth/no-modal) or
[yarn](https://yarnpkg.com/package/@web3auth/no-modal):

```bash npm2yarn
npm install --save @web3auth/no-modal @web3auth/base
```
