---
title: Whitelabeling
---

Web3Auth is fully whitelabelable with application branding to have a consistent user experience. You can pass your parameters while

- Instantiating Web3Auth to set the UI of the Modal (`uiConfig`)
- Instantiating Openlogin Adapter to customise the intermediary screens while login happens (`whiteLabel`)
- Initialising Web3Auth Modal to change the visibility and order of login methods. (`modalConfig`)

:::info

For more info on Whitelabeling refer to the [Whitelabeling Documentation](/pnp/features/whitelabel/).

:::
