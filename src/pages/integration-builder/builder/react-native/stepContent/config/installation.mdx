---
title: Install Web3Auth
---

Install the `@web3auth/react-native-sdk` using either [npm](https://www.npmjs.com/package/@web3auth/react-native-sdk) or
[yarn](https://yarnpkg.com/package/@web3auth/react-native-sdk):

```bash npm2yarn
npm install --save @web3auth/react-native-sdk
```

[![Npm package monthly downloads](https://badgen.net/npm/dm/@web3auth/react-native-sdk)](https://www.npmjs.com/package/@web3auth/react-native-sdk)

:::warning Update to the Latest-SDK

Please use the **[latest version](https://www.npmjs.com/package/@web3auth/react-native-sdk)** of Web3Auth SDKs.

:::
