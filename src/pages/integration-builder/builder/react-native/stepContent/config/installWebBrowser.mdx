---
title: Install TorusLabs WebBrowser
---

You will also require a WebBrowser implementation to allow our JS-based SDK to interact with the native APIs. For bare react native flow, we have made
our own implementation of the Expo Web Browser.

Install the `@toruslabs/react-native-web-browser` using either [npm](https://www.npmjs.com/package/@toruslabs/react-native-web-browser) or
[yarn](https://yarnpkg.com/package/@toruslabs/react-native-web-browser):

```bash npm2yarn
npm install --save @toruslabs/react-native-web-browser
```
