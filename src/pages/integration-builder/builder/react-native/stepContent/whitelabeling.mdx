---
title: Whitelabeling
---

Web3Auth is fully whitelabelable with application branding to have a consistent user experience. You can pass the `whiteLabel` your parameter in your
constructor to customise the intermediary screens while login happens.

:::info

For more info on Whitelabeling refer to the [Whitelabeling Documentation](/pnp/features/whitelabel/).

:::
