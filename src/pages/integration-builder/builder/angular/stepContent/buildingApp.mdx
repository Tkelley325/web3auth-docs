---
title: Using Web3Auth Angular Quick Start Guide
---

To use this Angular Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply
copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Community](https://web3auth.io/community/) for us to help you out as soon as
possible!

:::tip building locally

- Create a new angular project using the following command:

  ```shell
  ng new angular-web3auth-example
  ```

- In the `src/app/` folder, replace `app.components.ts`, `app.components.css`, `app.components.html` & add your blockchain specific `.ts` file.

- Run the app using

  ```bash npm2yarn
  npm run start
  ```

:::
