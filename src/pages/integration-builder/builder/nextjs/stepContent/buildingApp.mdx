---
title: Using Web3Auth NextJS Quick Start Guide
---

To use this NextJS Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply
copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Community](https://web3auth.io/community/) for us to help you out as soon as
possible!

:::tip building locally

- Create a new nextjs project using the following command:

  ```shell
  yarn create next-app --typescript
  ```

- Replace the file `package.json` in the root folder.
- In the `pages/` folder, add `App.tsx` & add your blockchain specific `.ts` file.
- In the `pages/` folder, replace `index.tsx`.
- Finally, replace the file `globals.css` in the `styles` folder.
- Run the app using

  ```bash npm2yarn
  npm run dev
  ```
