---
title: Using Web3Auth Vue Quick Start Guide
---

To use this Vue Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply copy
paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Community](https://web3auth.io/community/) for us to help you out as soon as
possible!

:::tip building locally

- Create a new vue project using the following command:

  ```shell
  vue create hello-world
  ```

  > In the terminal menu, select the option `Manually select features` and select `Typescript` from the menu. Use Vue `3.x` as the version and keep
  > everything else as default.

- In the `src/` folder, replace `Home.vue`, `App.vue` & add your blockchain specific `.ts` file.

- Run the app using

  ```shell npm2yarn
  npm run serve
  ```

:::
