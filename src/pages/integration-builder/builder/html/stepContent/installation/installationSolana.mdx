---
title: Install Solana Libraries
---

Install Solana Integration libraries using using a CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/@solana/web3.js@1.43.5/lib/index.iife.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@web3auth/solana-provider@latest/dist/solanaProvider.umd.min.j"></script>
```

:::tip

Add the file `solanaRPC.ts` to the project. This file contains the code to interact with the blockchain.

:::
