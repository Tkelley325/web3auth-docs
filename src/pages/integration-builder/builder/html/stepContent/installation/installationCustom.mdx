---
title: Install Web3Auth
---

Install `@web3auth/modal` & `@web3auth/openlogin-adapter` using a CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/@web3auth/modal@latest/dist/modal.umd.min.js"></script>
```

[![Npm package monthly downloads](https://badgen.net/npm/dm/@web3auth/modal)](https://www.npmjs.com/package/@web3auth/modal)
[![](https://data.jsdelivr.com/v1/package/npm/@web3auth/modal/badge)](https://www.jsdelivr.com/package/npm/@web3auth/modal)

This is the main Plug and Play Modal package that contains the Web3Auth SDK with a login UI, giving you a simple way of implementing Web3Auth within
your interface.

```html
<script src="https://cdn.jsdelivr.net/npm/@web3auth/openlogin-adapter@latest/dist/openloginAdapter.umd.min.js"></script>
```

[![Npm package monthly downloads](https://badgen.net/npm/dm/@web3auth/openlogin-adapter)](https://www.npmjs.com/package/@web3auth/openlogin-adapter)
[![](https://data.jsdelivr.com/v1/package/npm/@web3auth/openlogin-adapter/badge)](https://www.jsdelivr.com/package/npm/@web3auth/openlogin-adapter)

For using Custom Authentication & Whitelabeling, we need to use the OpenLogin Adapter, where we can initialise the authentication details and add
customisations in the login UI.

:::warning Update to the Latest-SDK

Please use the **[latest version](https://www.npmjs.com/package/@web3auth/modal)** of Web3Auth SDKs.

:::
