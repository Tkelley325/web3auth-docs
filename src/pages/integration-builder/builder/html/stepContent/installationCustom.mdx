---
title: Importing the required packages
---

Import the Web3Auth package using a CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/@web3auth/modal.umd.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@web3auth/openlogin-adapter@0/dist/openloginAdapter.umd.min.js"></script>
```

[![](https://data.jsdelivr.com/v1/package/npm/@web3auth/modal/badge)](https://www.jsdelivr.com/package/npm/@web3auth/modal)

**`@web3auth/modal`**

This is the main Plug and Play package that contains the Web3Auth SDK with a login UI, giving you a simple way of implementing Web3Auth within your
interface.

**`@web3auth/openlogin-adapter`**

For using Custom Authentication & Whitelabeling, we need to use the OpenLogin Adapter, where we can initialise the authentication details and add
customisations in the login UI.

> Additionally, import your respective blockchain interaction libraries and provider package highlighted here. Learn more about
> [Providers in Web3Auth](https://docs.web3auth.io/sdk/helper-sdks/providers/).
